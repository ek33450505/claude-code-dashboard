Plumbob Marker Style

When an agent goes active, a marker floats above their sprite. Which style fits the CAST HQ vibe?

A

Pixel Diamond

Classic Sims plumbob — 5×5 pixel grid diamond, steps() animation, pure 8-bit. Floats in accent color above sprite.

accent color per agent

Pros

  • Most authentic Sims reference
  • Fits 8-bit aesthetic
  • steps() = crisp pixel motion

Cons

  • Small at scale=2 sprites
  • More CSS to maintain
B

Rotating Diamond

CSS clip-path rotated square, smooth float + slow rotation. More modern/polished than pixel grid.

smooth ease-in-out + hue shift

Pros

  • Clean, bold shape
  • Readable at any scale
  • Glow looks great on dark bg

Cons

  • Smooth animation breaks 8-bit feel
  • Less "Sims" reference
C

Pixel Chevron

▲ or ◆ unicode character with text-shadow glow, steps() bounce. Ultra-simple, zero extra markup.

◆
unicode + text-shadow glow

Pros

  • Lowest complexity
  • Works at any scale
  • Easy to color per-agent

Cons

  • Least unique / least "Sims"
  • Font rendering varies by OS
D

Pulsing Orb

Small glowing circle with an expanding pulse ring — like a radar ping. Float + ring-pulse animation.

float + expand pulse ring

Pros

  • Feels like a live signal/beacon
  • Fits "war room" concept
  • Reuses existing pulse keyframe

Cons

  • Not Sims-style at all
  • Less distinctive
